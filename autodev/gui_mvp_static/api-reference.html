Canonical Routes

The GUI consumes these routes directly. Saved compare snapshot item mutations should use the canonical PATCH and DELETE resource routes.

Run Detail

GET /api/runs
List dashboard runs.
GET /api/runs/<run_id>
Load selected run detail and trust context.
GET /api/runs/<run_id>/artifacts/read
Read `.autodev` artifacts in the Validation tab.

Trust Intelligence

GET /api/autonomous/quality-gate/latest
Latest autonomous quality-gate summary.
GET /api/autonomous/trust/latest
Latest trust summary and full trust packet.
GET /api/autonomous/trust/trends
Historical trust trend summary.

Compare Snapshots

GET /api/runs/compare
Compare two runs across trust, validation, and execution metrics.
POST /api/runs/compare/snapshots
Persist a compare payload as a saved snapshot.
GET /api/runs/compare/snapshots
List saved snapshots with query, tag, date, pinned, archive, and pagination filters.
GET /api/runs/compare/snapshots/<snapshot_id>
Open one saved snapshot.
PATCH /api/runs/compare/snapshots/<snapshot_id>
Canonical route for rename and metadata updates.
DELETE /api/runs/compare/snapshots/<snapshot_id>
Canonical route for deleting a saved snapshot.
POST /api/runs/compare/snapshots/import
Import an exported compare snapshot payload.
POST /api/runs/compare/snapshots/bulk
Bulk metadata updates or deletes.
POST /api/runs/compare/snapshots/retention/apply
Preview or apply retention cleanup.

Deprecated Helpers

POST /api/runs/compare/snapshots/<snapshot_id>/rename
Deprecated helper. Use PATCH /api/runs/compare/snapshots/<snapshot_id>.
POST /api/runs/compare/snapshots/<snapshot_id>/metadata
Deprecated helper. Use PATCH /api/runs/compare/snapshots/<snapshot_id>.
POST /api/runs/compare/snapshots/<snapshot_id>/delete
Deprecated helper. Use DELETE /api/runs/compare/snapshots/<snapshot_id>.
Legacy helper responses include deprecation metadata and canonical replacement guidance.
